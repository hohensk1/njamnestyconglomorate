# README
Problem description:
I am on the e-board for the TCNJ branch of Amnesty International
They work to protect and defend against human rights violations. 
We are the first official college-level institution in New Jersey. There are a couple of other colleges attempting to set up clubs.
We are working to form a conglomerate between these clubs to share information, resources, and plan events together.

Rationale or motivation for choosing this problem:
I hope that this program contributes to the spread of information about these current crises that people might be unaware of. The spread of this information will enable us to more easily identify the signs of a humanitarian crisis and provide us with greater access to resources like partitions and other means of direct assistance to those in need.

Overview of the proposed functionality and how a need is met:
My goal is to create an application that allows this conglomerate to share information about humanitarian crises. 
Community calendar for large events going on and events that the different clubs are running
Also, private events that are only going on within the college
Area to post slide shows
Area to post partitions
Area to post about the humanitarian crisis currently going on 
Which can sort by the current crisis
A section for local information
Each club branch would have an account
